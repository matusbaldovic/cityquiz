# 🌍 City Quiz

City Quiz is a desktop application developed in **Python** using the **Tkinter** library. The player enters their personal information and answers a series of questions about world capitals. At the end of the quiz, the final score is displayed and saved to a CSV file.

## ✨ Features

- User registration (name, age, city)
- Graphical user interface built with Tkinter
- Multiple-choice quiz questions
- Automatic score calculation
- Final score screen
- Results saved to `vysledky.csv`
- Custom images and simple UI design

## 🛠 Technologies Used

- Python 3
- Tkinter
- Pillow (PIL)
- CSV

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/matusbaldovic?tab=repositories
```

2. Navigate to the project folder:

```bash
cd cityquiz
```

3. Install the required dependency:

```bash
pip install pillow
```

4. Run the application:

```bash
python test.py
```
